# Part 1 – Hardware vs Software

## Explain Hardware
Write **at least 50 words** explaining what hardware is.
Hardware refers to the physical, tangible parts of a computer or digital device that you can see and touch. Examples include the CPU, RAM, motherboard, storage drives, keyboard, mouse, monitor, printer, and network cards. Hardware performs the actual processing, storage, input, output, and communication tasks. Without hardware, software has no physical platform on which to run. Hardware components work together through electrical signals and standardized interfaces to execute instructions and handle data.

## Explain Software
Write **at least 50 words** explaining what software is.
Software is the collection of programs, instructions, and data that tells hardware what to do. Examples include operating systems such as Windows, macOS, and Linux, application programs such as web browsers, word processors, and games, and utility programs such as antivirus tools. Software is intangible—you cannot physically touch it like hardware. It provides the logic, rules, and user interface that allow people to interact with and control computer hardware effectively.

## How Do Hardware and Software Interact?
Write **at least 50 words** describing how they interact.
Hardware and software depend on each other. Software sends instructions that the hardware executes. For example, when you open a web browser, the operating system loads the browser program from storage into RAM, and the CPU processes its instructions. Input hardware such as the keyboard or mouse sends signals to the software, while output hardware such as the monitor displays the results generated by the software. The operating system acts as an intermediary, managing communication between applications and hardware devices through drivers and system services. Without software, hardware would have no instructions to follow; without hardware, software would have no physical platform on which to run.
