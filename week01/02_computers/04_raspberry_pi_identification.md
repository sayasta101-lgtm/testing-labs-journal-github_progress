| Component | Function |
|-----------|----------|
| SoC |The main processing chip that combines the CPU, GPU, memory controller, and other functions into a single integrated circuit. |
| GPIO |Pins that low the device to interact with external hardware such as LEDs, sensors, buttons, and motors. |
| USB |Used to connect peripherals such as keyboards, mice, storage devices, and other accessories. |
| HDMI |Provides digital video and audio output for connecting the device to a monitor, TV, or projector. | 
| Power |Supplies electrical energy required for the device to operate. | 
| microSD |A removable storage card used for the operating system, applications, and user data. |
| Network |Provides wired network connectivity, usually through an Ethernet port, for internet and local network access. |
| Wireless |Enables wireless communication through technologies such as Wi-Fi and Bluetooth for internet access and device connections. |

## Reflection
Why are many components integrated into a single board in embedded computers such as the Raspberry Pi?
Write **100 words**.
Many components are integrated into a single board in embedded computers such as the Raspberry Pi to make the system smaller, cheaper, and more energy efficient. Combining components like the processor, memory interfaces, networking, and input/output controllers on one board reduces the number of separate parts required. This improves reliability because there are fewer connections that can fail. Integration also simplifies manufacturing and assembly, lowering production costs. Additionally, a single-board design uses less power and generates less heat, making it suitable for portable and embedded applications. These advantages make devices like the Raspberry Pi compact, affordable, efficient, and easy to use.
