| Component | Function | Key Specification |
|----------|---------|------------------|
| CPU | Executes instructions and performs calculations required by programs and the operating system.| Clock Speed (GHz), Number of Cores/Threads, Cache Size|  
| RAM | Temporarily stores data and programs currently in use for quick access by the CPU.| |Capacity (GB), Speed (MHz/MT/s), DDR Type (e.g., DDR4, DDR5) 
| Storage | Permanently stores the operating system, applications, and user files| Capacity (GB/TB), Type (SSD/HDD), Read/Write Speed| .
| Motherboard | Connects and allows communication between all computer components.| Chipset, Socket Type, Form Factor (ATX, Micro-ATX, Mini-ITX)|
| Power Supply | Converts mains electricity into usable power for computer components.|  Wattage (W), Efficiency Rating (80 Plus Bronze/Gold, etc.)|
| Cooling |  Removes heat from components to maintain safe operating temperatures.||Cooling Type (Air/Liquid), Fan Size (mm), Airflow (CFM)
