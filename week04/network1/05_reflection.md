## 🧠 5. Reflection Questions

### 5.1 What would happen if two PCs had the same IP address?
_Write answer…_ If two PCs have the same IP address on the same network, it causes an IP conflict. The network becomes confused about which device should receive data, leading to connection errors, dropped internet access, or one/both devices losing network connectivity.

### 5.2 Why do we use a switch instead of directly connecting PCs together?
_Write answer…_ A switch allows multiple devices to communicate efficiently within a network. Unlike directly connecting PCs, a switch manages data traffic, reduces collisions, supports many devices at once, and improves speed and network organization

### 5.3 What does the subnet mask 255.255.255.0 indicate?
_Write answer…_ It means the first three parts (octets) of the IP address identify the network, and the last part identifies individual devices (hosts). It allows up to 254 usable devices in that network (from .1 to .254).
