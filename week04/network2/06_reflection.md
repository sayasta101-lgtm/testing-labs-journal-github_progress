
## 🧠 6. Reflection

Answer in full sentences:

1. What are the advantages of DHCP over static IP assignment? = DHCP (Dynamic Host Configuration Protocol) automatically assigns IP addresses to devices on a network. This reduces manual work and avoids configuration errors. It also prevents IP conflicts because each device receives a unique address automatically. DHCP is easier to manage, especially in large networks where devices frequently join or leave.
3. How does DNS improve usability in a network? = DNS (Domain Name System) makes networks easier to use by converting domain names (like www.google.com) into IP addresses that computers understand. This means users don’t need to remember long numbers; they can simply use names to access websites and services. 
4. What would happen if the DHCP server went offline?  = If the DHCP server goes offline, new devices will not be able to automatically receive IP addresses. Existing devices may continue working temporarily using their current IP addresses, but if their lease expires, they could lose network connectivity. This can cause network disruptions

The more challenging part is often configuring IP addressing and services like DHCP/DNS correctly, since small mistakes can stop devices from communicating.
